## Sainsbury’s Scraper

## System Requirement

Web server running PHP 5.0 or higher with cURL extension and composer installed

## Installation

$ git clone https://github.com/file3/to_scraper.git

$ cd to_scraper

$ composer update

## Running

$ php index.php

## Test

$ vendor/bin/phpunit tests/GetDataTest
